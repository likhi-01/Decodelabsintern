# Number Game 🎮

## Project Overview

The Number Game is a Java console-based application that generates a random number between 1 and 100 and challenges the user to guess it. The program provides hints indicating whether the guess is too high or too low until the correct number is guessed.

## Features

* Random number generation
* User input handling using Scanner
* High/Low feedback for each guess
* Attempt counter
* Interactive console-based gameplay

## Technologies Used

* Java
* Scanner Class
* Random Class
* Loops (while loop)
* Conditional Statements (if-else)

## How to Run

1. Clone the repository.
2. Open the project in IntelliJ IDEA, Eclipse, or VS Code.
3. Compile the program:
   javac Numbergame.java
4. Run the program:
   java Numbergame
